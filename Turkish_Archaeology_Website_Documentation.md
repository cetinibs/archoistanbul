# Turkish Archaeology Website - ArcheoTR

## Project Overview

This project is a professional Turkish archaeology website created based on research of Polish archaeology websites, particularly modeling after archeopol.com and other similar sites. The website serves as a comprehensive platform for archaeological services in Turkey.

## Research Foundation

### Polish Archaeology Websites Analyzed:
- **archaeolink.com**: Educational resource with 40,000+ archaeology resources
- **archeowiesci.pl**: Polish archaeology news from University of Warsaw
- **nadzory-archeologiczne.pl**: Commercial archaeology services
- **archeolog.info**: Professional archaeological services with 20+ years experience
- **archeoplus.ch**: Swiss archaeology with 3D graphics and restorations

### Common Features Identified:
- Archaeological excavations and surveys
- Documentation and research services
- Professional services for investors
- Cultural heritage protection
- Educational resources
- Museum and restoration services

## Technical Setup

### To Run the Website:
1. Ensure all files are in the same directory
2. Start a local web server: `python3 -m http.server 8000`
3. Open browser to `http://localhost:8000`

### Dependencies:
- Google Fonts (Roboto, Playfair Display)
- Font Awesome icons
- Modern web browser with CSS3 and ES6 support
