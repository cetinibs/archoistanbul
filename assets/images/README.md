# Görsel Dosyaları

Bu klasörde aşağıdaki görsel dosyaları bulunmalıdır:

## Gerekli Dosyalar:

1. **logo.png** - Archeopol logosu (önerilen boyut: 200x60px, transparent PNG)

2. **hero-bg.jpg** - Ana sayfa hero bölümü arka plan görseli (önerilen boyut: 1920x1080px, optimize edilmiş JPEG)

3. **favicon.ico** - Site ikonu (16x16px ve 32x32px boyutlarında)

4. **og-image.jpg** - Social media paylaşımları için görsel (önerilen boyut: 1200x630px)

## Görsel Optimizasyonu:

- Tüm görseller web için optimize edilmelidir
- JPEG dosyaları için %85 kalite önerilir
- PNG dosyaları için TinyPNG kullanılabilir
- WebP formatı da desteklenmektedir

## Lisans:

Tüm görseller telif hakkına tabi olabilir. Kullanmadan önce lisans kontrolü yapınız.
