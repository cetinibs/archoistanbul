# Archeopol - Arkeoloji Kaynakları ve Araştırma Merkezi

## Genel Bakış
Archeopol, arkeoloji, antropoloji ve antik medeniyetler hakkında kapsamlı bilgi sunan modern ve hızlı bir web sitesidir.

## Özellikler
- 📱 Mobil uyumlu responsive tasarım
- ⚡ Hızlı yükleme süreleri
- 🔍 SEO optimizasyonu
- 🌍 Global CDN desteği
- 🔒 Güvenlik başlıkları
- 💾 Offline çalışma desteği (Service Worker)
- 🎨 Modern ve kullanıcı dostu arayüz

## Teknolojiler
- HTML5
- CSS3 (CSS Grid, Flexbox)
- Vanilla JavaScript
- Service Worker
- Font Awesome Icons
- Google Fonts

## Kurulum
1. Dosyaları web sunucunuza yükleyin
2. HTTPS sertifikası kurulumu yapın
3. .htaccess dosyasının çalıştığından emin olun

## Performans Optimizasyonları
- Gzip sıkıştırma
- Browser önbellekleme
- Lazy loading görüntüler
- Minified CSS/JS
- Service Worker ile offline destek

## SEO
- Meta etiketleri
- Open Graph protokolü
- Sitemap.xml
- Robots.txt
- Yapılandırılmış veri desteği

## Güvenlik
- HTTPS zorunluluğu
- XSS koruması
- Clickjacking koruması
- Content Security Policy
