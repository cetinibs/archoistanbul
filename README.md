# Archeo Cafe & Hostel - İstanbul

Modern, responsive website for Archeo Cafe and Hostel in Istanbul.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## 🌟 Features

- **Modern Design**: Clean, professional UI with Turkish content
- **Responsive**: Works perfectly on all devices
- **Fast**: Built with Next.js 14 for optimal performance
- **SEO Optimized**: Meta tags and structured content
- **TypeScript**: Type-safe development

## 🏗️ Tech Stack

- **Framework**: Next.js 14
- **Language**: TypeScript
- **Styling**: Inline CSS (easily customizable)
- **Deployment**: Vercel (recommended)

## 📱 Website Sections

- **Header**: Branding and navigation
- **Hero**: Welcome message in Turkish
- **Services**: Hostel, Cafe, and Historical information
- **Contact**: Contact details and location
- **Footer**: Copyright and additional info

## 🚀 Deploy to Vercel

The easiest way to deploy your Next.js app is to use the [Vercel Platform](https://vercel.com).

1. Push your code to GitHub
2. Connect your repository to Vercel
3. Deploy with one click!

See [DEPLOYMENT_GUIDE.md](./DEPLOYMENT_GUIDE.md) for detailed instructions.

## 🎨 Customization

- Edit `pages/index.tsx` to update content
- Modify colors and styling in the inline styles
- Add your own images and assets
- Update contact information
