# ArcheoTR - Turkish Archaeology Website

A professional Turkish archaeology website created based on research of Polish archaeology sites, particularly modeling after archeopol.com and similar professional archaeological service providers.

## Quick Start

1. Open `index.html` in a web browser, or
2. Run a local server: `python3 -m http.server 8000`
3. Visit `http://localhost:8000`

## Features

- Professional archaeological services presentation
- SEO-optimized Turkish content
- Mobile-responsive design
- Interactive gallery and project showcase
- Contact forms and professional branding

Professional archaeological services including excavation, research, documentation, restoration, consulting, and education in Turkey.
